YouTube Scraper API Documentation

Simple API to search and download YouTube videos

Endpoints

1. Search Videos

GET /search

Search for YouTube videos based on a query.

Query Parameters:
  • query (required): The search term. Example: ?query=music
Example Request:
GET /search?query=music
Response:
{
  "success": true,
  "data": [
    {
      "title": "Example Video",
      "url": "https://www.youtube.com/watch?v=ID",
      "videoId": "ID",
      "duration": "3:45",
      "views": "1M views",
      "uploaded": "1 day ago",
      "thumbnail": "https://i.ytimg.com/vi/ID/hqdefault.jpg",
      "channel": {
        "name": "Channel Name",
        "url": "https://www.youtube.com/channel/ID"
      }
    }
  ]
}

2. Download Video/Audio

GET /download

Download YouTube videos or audio files.

Query Parameters:
  • url (required): The YouTube video URL. Example: ?url=https://youtube.com/watch?v=ID
  • type (optional): The type of content to download (audio or video). Default: video
  • quality (optional): The quality of the video (144p, 240p, 360p, etc.). Default: 720p
Example Request:
GET /download?url=https://youtube.com/watch?v=ID&type=video&quality=720p
Response:
{
  "success": true,
  "data": {
    "title": "Example Video",
    "duration": "3:45",
    "thumbnail": "https://i.ytimg.com/vi/ID/hqdefault.jpg",
    "downloadUrl": "https://download-link",
    "quality": "720p",
    "type": "video"
  }
}
